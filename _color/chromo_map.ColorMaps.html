chromo_map.ColorMaps

class chromo_map.ColorMaps(*args, **kwargs)[source]

A class for collecting color gradients.

This class is a dictionary of color gradients. You can access the gradients by their names or by their categories.

Examples

Access a color gradient by name:

from chromo_map import cmaps
cmaps.plotly.carto.Vivid
