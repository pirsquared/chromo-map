chromo_map.ColorGradient.with_alpha

ColorGradient.with_alpha(alpha, name=None)[source]

Create a new gradient with a new alpha value.

Parameters:

  • alpha (float) – The new alpha value between 0 and 1.

  • name (str, optional) – The name of the new gradient.

Returns:

The new gradient with the new alpha value.

Return type:

ColorGradient

Examples

Create a gradient with a new alpha value:

from chromo_map import ColorGradient
colors = ['#ff0000', '#00ff00', '#0000ff']
gradient = ColorGradient(colors, name='rGb')
gradient.with_alpha(0.5)
