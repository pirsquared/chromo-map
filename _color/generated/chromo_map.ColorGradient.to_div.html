chromo_map.ColorGradient.to_div

ColorGradient.to_div(maxn=None, as_png=False)[source]

Convert the gradient to an HTML div.

Parameters:

  • maxn (int, optional) – The maximum number of colors to display.

  • as_png (bool, optional) – Whether to display the gradient as a PNG image.

Returns:

The gradient as an HTML div.

Return type:

HTML

Examples

Convert the gradient to an HTML div:

from chromo_map import ColorGradient
colors = ['#ff0000', '#00ff00', '#0000ff']
gradient = ColorGradient(colors, name='rGb')
gradient.to_div(as_png=False)
rGb
